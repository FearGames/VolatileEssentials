# VolatileEssentials

A simple plugin that alters the Essentials user persistence system.

Currently supported modes:
- MongoDB backend

### Requirements
- Spigot/Paper 1.12.2+
- EssentialsX 2.17.1+
- JDK 8 (doesn't work on java 9+), JRE isn't supported!
